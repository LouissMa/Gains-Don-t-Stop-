# 摘要

针对互联网电影资源快速增长所带来的信息过载问题，本文研究基于神经协同过滤的个性化电影推荐方法。实验采用MovieLens 100K公开数据集，将评分大于或等于4分的记录转换为用户正反馈，并按照时间顺序采用留一法划分训练集、验证集和测试集。在统一的数据划分、负采样和评价协议下，本文实现并比较了广义矩阵分解模型GMF、多层感知机模型MLP和神经矩阵分解模型NeuMF，使用HR@5、NDCG@5、HR@10、NDCG@10及采样AUC评价推荐性能。实验结果表明，NeuMF在各项测试指标上均取得最优结果，其中HR@10为0.636943，NDCG@10为0.385326，采样AUC为0.884825。与MLP的NDCG@10（0.368987）相比，NeuMF绝对提高0.016339，相对提高约4.43%，说明融合线性交互和非线性交互能够小幅改善目标电影的排序位置。在此基础上，本文加载训练完成的NeuMF模型，排除用户已经评分的电影，对其余候选电影进行打分和排序，生成Top-10个性化推荐列表。实验保留了原始数据哈希、训练配置、逐轮日志、模型检查点和测试结果，保证了实验过程的真实性与可复现性。研究结果验证了神经协同过滤方法在个性化电影推荐任务中的有效性。

**关键词：** 个性化推荐；电影推荐；神经协同过滤；NeuMF；深度学习
