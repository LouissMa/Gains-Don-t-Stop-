# 论文方向与评分标准最终审查

## 一、审查结论

论文题目“基于神经协同过滤的个性化电影推荐方法研究”属于具体的深度学习应用任务，研究对象、模型输入输出、训练过程、评价协议和推荐结果均围绕个性化电影推荐展开，没有跑题。

本文不是普通电影管理系统，也不是纯文献综述。核心工作是使用GMF、MLP和NeuMF学习用户与电影之间的匹配关系，通过真实实验比较模型，并使用最优模型从用户未评分电影中生成Top-10推荐列表。因此，“模型对比”是验证推荐算法有效性的实验手段，“Top-N推荐列表”是算法的实际任务输出，两部分共同支撑论文题目。

## 三、方向正确的关键证据

1. 任务输入是用户历史评分及候选电影编号。
2. 任务输出是按模型匹配分数排序的Top-N电影列表。
3. 训练目标是学习用户与电影的隐式反馈关系，而不是实现电影增删改查。
4. 三种模型均属于神经协同过滤框架，NeuMF是主要深度学习模型。
5. 测试指标HR、NDCG和AUC直接评价推荐排序效果。
6. 推荐程序排除用户已评分电影，对全部剩余电影打分并输出推荐结果。
7. 实验保存了训练日志、模型检查点、指标CSV、环境信息和哈希，可核验真实性。
