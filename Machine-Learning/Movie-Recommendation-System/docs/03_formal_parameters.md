# 正式实验参数确认（第8部分）

以下参数已于2026年6月7日确认，作为后续MovieLens 100K正式实验的统一配置。

## 数据与任务

| 参数 | 正式取值 |
|---|---:|
| 正反馈阈值 | 评分大于等于4 |
| 最少正反馈数 | 3 |
| 数据划分 | 按时间留一法 |
| 每条训练正样本的负样本数 | 4 |
| 每位用户的评估负样本数 | 99 |

训练负样本排除用户所有已评分电影。验证集和测试集候选列表分别使用固定随机种子生成，且三个模型共享完全相同的候选列表。

## 模型

| 参数 | 正式取值 |
|---|---|
| 对比模型 | GMF、MLP、NeuMF |
| 嵌入维度 | 32 |
| MLP隐藏层 | 64、32、16、8 |
| 输出 | 单个未激活logit |

## 训练

| 参数 | 正式取值 |
|---|---:|
| 最大Epoch | 30 |
| Batch Size | 4096 |
| 学习率 | 0.001 |
| 优化器 | Adam |
| 损失函数 | BCEWithLogitsLoss |
| Early Stopping耐心值 | 5 |
| 随机种子 | 2026 |
| DataLoader进程数 | 0 |

最大Epoch为30，但如果验证集NDCG@10连续5轮没有提高，训练将提前结束。最优模型由验证集NDCG@10确定，测试集只在该模型训练结束后评估一次。

## 评价

| 类型 | 指标 |
|---|---|
| 主要指标 | NDCG@10 |
| 辅助指标 | HR@5、NDCG@5、HR@10、采样AUC |

## 正式训练保护

MovieLens 100K最终30轮训练已经完成。为防止误运行，当前配置重新关闭训练许可：

```toml
formal_training_approved = false
allow_overwrite_formal_outputs = false
```

正式训练必须满足以下全部条件：

1. 全部自动测试再次通过；
2. 明确检查GPU、数据哈希和正式输出目录；
3. 将`formal_training_approved`改为`true`；
4. 使用带`--confirm-formal-training`的正式命令；
5. 正式输出目录为空，且默认禁止覆盖。
